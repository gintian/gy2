世纪eHR4客户端注意事项

1. 对客户端系统的要求
1.1 操作系统的版本要求

需要Windows9x/NT4.0SP6/XP/2000，任选；建议安装所有最新的更新及安全补丁。

1.2 IE的版本要求

需要IE6.0版本及以上，并升级至SP1，建议安装所有最新的更新及安全补丁。

2.客户端IE的设置

2.2 将WEB服务器地址加入安全站点：

由于eHR4中使用了AcitveX控件(客户端打开EXCEL、WORD文档等)，而IE浏览器一般情况下会对AcitveX控件进行拦截，造成无法正常使用eHR4系统。为了解决这个问题，可以将WEB服务器的域名或地址（例如 http://192.192.100.102:8080）加入到IE浏览器所信任的站点。

2.2.1 设置安全站点：

打开位置：打开IE工具Internet选项“安全”标签页受信任的站点站点
打开：设置安全站点
将“对该区域中的所有站点要求服务器验证(https:)”复选框的勾去掉。在输入框中输入：eHR4的WEB服务器的域名ttp://192.192.100.102:8080后，点击“添加”按钮。
设置安全站点
2.2.2 启用ActiveX控件：

打开位置：打开IE工具Internet选项“安全”标签页受信任的站点“自定义级别”
打开：设置“自定义级别”
选择启用“ActiveX控件、Javas和插件、脚本
3.客户端IE相关工具（或插件）的设置 卸载IE相关插件（如：上网助手等,在已启用的列表选中这些扩展，然后再按禁用即可），或设置不拦截弹出窗口

设置不拦截弹出窗口
4.Windows XP SP2和相关IE插件的设置
输入eHR4的WEB服务器的域名或IP地址，设置可以弹出窗口
5.其它工具栏：
6.检查所存网页较新版本设置,打开位置：打开IE工具Internet选项“常规”,点击设置按钮

8.后台业务下载

对于eHR4业务平台，需要设置Windows XP SP2和其它IE插件允许从Web服务器下载后台业务。当提示是否安装后台业务时，选择‘是’即可。
